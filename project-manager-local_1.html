🔗 連結 Google 日曆

完成設定後，你在 Google 日曆新增的事件會自動出現在這裡，這裡新增的事件也會同步到 Google 日曆。

1
開啟 Google Cloud Console
前往 console.cloud.google.com，用你的 Google 帳號登入，建立新專案（名稱隨意）。
2
啟用 Google Calendar API
在左側選單 →「API 和服務」→「程式庫」→ 搜尋「Google Calendar API」→ 點「啟用」。
3
建立 API 金鑰
「API 和服務」→「憑證」→「建立憑證」→「API 金鑰」，複製金鑰。
4
貼入下方並填入你的日曆 ID
日曆 ID 在 Google 日曆設定 → 選擇日曆 → 向下捲動找到「日曆 ID」，通常是你的 Gmail 地址。
