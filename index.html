🔗 連結 Google 日曆

填入 API 金鑰和日曆 ID，即可把 Google 日曆的事件自動匯入本 App，並保持同步。

1
啟用 Google Calendar API
前往 Google Cloud Console → API 和服務 → 程式庫 → 搜尋「Google Calendar API」→ 啟用
2
建立 API 金鑰
API 和服務 → 憑證 → 建立憑證 → API 金鑰，複製金鑰貼入下方
3
將日曆設為公開
Google 日曆 → 日曆設定 → 存取權限 → 勾選「公開提供這個日曆」
4
填入日曆 ID
日曆設定 → 向下捲動找「日曆 ID」，通常是你的 Gmail 地址
